# Code of Conduct — Dashboard MRO

Tous les contributeurs doivent adhérer au code de conduite suivant.

---

## 1. Respect mutuel

- Aucun harcèlement, discrimination, intimidation, ou agressivité.
- Les débats scientifiques doivent rester courtois.

---

## 2. Conduite scientifique

- Pas de plagiat
- Attribution correcte des idées
- Reproductibilité
- Transparence des méthodes

---

## 3. Collaboration constructive

- Suggérer est mieux que dénigrer
- Proposer des alternatives
- Documenter les choix

---

## 4. Sécurité

- Pas d’injection de code malveillant
- Pas de dépendances dangereuses
- Pas d’exécution arbitraire côté client

---

## 5. Signalement

Tout comportement problématique peut être signalé via :
**issues GitHub** (#report)

---

En contribuant, vous acceptez de suivre ces règles.
